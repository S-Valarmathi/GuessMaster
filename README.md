# 🎯 GuessMaster

An interactive number guessing game developed using HTML, CSS, and JavaScript.

## ✨ Features

- 🎯 Random number generation between 1 to 10
- 🏆 Dynamic score tracking system
- 🎉 Instant feedback with interactive messages
- 💻 Responsive and user-friendly interface
- 🌙 Modern glassmorphism design

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

## 🚀 How to Play

1. Enter a number between 1 and 10
2. Click the Check button to submit your guess
3. Get instant feedback and try to achieve the highest score 🎯

## 📌 Project Overview

A frontend mini-project created to practice JavaScript logic, DOM manipulation, and interactive web development concepts.
